All Features

  • Advanced Markdown FeaturesBold, Italic, Strikethrough, Headers, and way more...
  • PluginsNative plugin support, with 1 built-in plugin.
  • Custom ThemesYou can customize the interface using CSS or the client theme feature.
  • Chat FeaturesSlash Commands, Replies, Reactions, and more.
  • PerformanceSparkleCord can run on potato PCs.
  • File UploadsYou can upload any type of file, without size limits.
  • Accessibility FeaturesHigh Contrast Mode, Saturation, Link Underlining, and more.
  • Highly CustomizableYou can change many things using just the settings menu.

Frequently Asked Questions

Can you chat with other people?

Not yet. Self-hosting features will be implemented in the near future.

Is my data secure?

Absolutely. SparkleCord is fully client-side and open-source, so everything is on your device only.

Can I make custom plugins?

Yes, you can create a .js file in the /plugins subfolder, and the rest is simple.
